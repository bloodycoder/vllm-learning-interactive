# vllm-learning-interactive

Interactive React learning page for nano-vllm / KV Cache.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

This repo includes `.github/workflows/deploy.yml`.

After pushing to `main`, enable GitHub Pages:

1. Open repo settings.
2. Go to **Pages**.
3. Under **Build and deployment**, choose **GitHub Actions**.
4. Wait for the deploy workflow to finish.

The public URL should be:

```text
https://bloodycoder.github.io/vllm-learning-interactive/
```
